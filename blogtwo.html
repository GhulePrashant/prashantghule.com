How to code in JavaScript?

JavaScript is a programming language used to make dynamic and interactive websites.

For example, it allows us to:

  • Show a message when the user clicks on a button.

  • Hide a message when the user clicks on the close button.

  • Open a sidebar when the users swipes to the right from the edge of the screen.

  • Hide or unhide the elements.

JavaScript is often used in combination with HTML & CSS.

JavaScript can also be used on the server side to write the backend part or API of your website. Then, in that context it is called Node or NodeJS.

Basics of JavaScript

Let's start with the basics by looking at a example:

        
            function sum(a,b){
            

            return a + b;

Return keyword

The keyword return instructs the function that this is the result of that function.

Strings

You can create a string in JavaScript by simply using double quotes ("") or single quotes (''). 

        
            "This is a string";

            'This is also a string!'

Basic String methods

Here are some methods you can call on strings:

.includes(searchString)

This method returns true when the searchString is included inside the parent string.
    "Prashant.includes("shant"); //true

    "Prashant".includes("gogo"); //false

.toUpperCase()

This method will return a new string that has all of its characters in upper case.
    "Prashant.toUpperCase(); //"PRASHANT"

.toLowerCase()

This method will return a new string that has all of its characters in lower case.
    "Prashant.toLowerCase(); //"prashant"
